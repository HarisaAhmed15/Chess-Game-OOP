## Project Description
This is a console-based Chess Game developed using Object-Oriented Programming concepts in C++.

## Features
- Complete chess board
- Piece movement validation
- Turn-based gameplay
- Win detection
- Console interface

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Polymorphism
- Abstract Classes
- Dynamic Memory Allocation
- Virtual Functions
